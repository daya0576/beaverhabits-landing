# Beaver Habits Landing Page

A modern, lightweight landing page for Beaver Habit Tracker SaaS built with Astro 5, React Islands, and Tailwind CSS 4.

## Tech Stack

- **Astro 5** - Static site generator with zero-JS by default
- **React 19** - Interactive components using Islands architecture
- **Tailwind CSS 4** - Modern CSS framework with native support
- **TypeScript** - Type-safe development

## Getting Started

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Commands

All commands are run from the root of the project:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
